🚀 Install ke Tampermonkey

Script akan di-download sebagai file .user.js. Tampermonkey akan otomatis detect dan tampilkan dialog install.

  1. File .user.js akan ter-download otomatis
  2. Tampermonkey akan otomatis detect file tersebut
  3. Dialog install Tampermonkey akan muncul → klik "Install"
  4. Selesai! Script aktif di browser Anda
💡 Catatan: Jika Tampermonkey tidak otomatis detect, buka Tampermonkey Dashboard → tab Utilities → Import dari file yang sudah di-download.
