Browser memblokir akses ke TamperMonkey. Gunakan cara manual berikut:

📋 Cara 1: Copy & Paste Manual
  1. Klik tombol "Copy Code" di editor
  2. Buka TamperMonkey dari icon browser
  3. Klik "Create a new script"
  4. Paste code (Ctrl+V) dan simpan
🔧 Cara 2: Matikan Ad Blocker
  1. Buka chrome://extensions/
  2. Cari ad blocker (uBlock, AdBlock, dll)
  3. Matikan atau whitelist TamperMonkey
  4. Refresh halaman ini
