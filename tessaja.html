⚙️ Setup Firebase (Sekali Saja)

Untuk menyimpan data online, buat project Firebase gratis:

  1. Buka console.firebase.google.com
  2. Klik "Add Project" → beri nama → Create
  3. Pilih "Build" → "Realtime Database" → Create Database
  4. Pilih location (asia-southeast1) → Start in test mode
  5. Klik ⚙️ Project Settings → scroll ke "Your apps" → pilih icon </> (Web)
  6. Copy firebaseConfig object, paste di bawah ini
⚠️ Config disimpan di browser Anda saja (localStorage). Data script tersimpan di Firebase cloud, bisa diakses dari perangkat manapun.
