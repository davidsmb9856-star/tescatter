Dashboard

Setup JSONBin API (Sekali Saja)

  1. Buka jsonbin.io/signup dan daftar gratis (bisa pakai Google).
  2. Setelah login, buka Dashboard → API Access.
  3. Copy X-Master-Key (format: $2a$10$xxxxx).
  4. Paste di bawah ini, lalu klik Simpan & Hubungkan.

API key hanya disimpan di browser Anda (sessionStorage). Data script tersimpan aman di cloud JSONBin.

📦 Apa itu TM Script Vault?

Aplikasi web untuk menyimpan script Tampermonkey/Greasemonkey Anda di cloud (JSONBin.io). Bukan Google Sheet, bukan localStorage PC. Semua pengguna bisa akses dari perangkat manapun.

🚀 Cara Mulai

  1. Daftar gratis di jsonbin.io
  2. Copy X-Master-Key dari dashboard
  3. Paste di halaman Dashboard, klik "Simpan & Hubungkan"
  4. Mulai simpan script Anda!

✨ Fitur

  • ✅ Simpan script tanpa limit karakter
  • ✅ Cloud storage (JSONBin) - akses dari mana saja
  • ✅ Search & filter script
  • ✅ Tag & versi script
  • ✅ Copy / Download sebagai .user.js
  • ✅ Backup & Restore JSON
  • ✅ Dark grey modern UI

🔒 Privasi

API key Anda hanya disimpan di sessionStorage browser (hilang saat tab ditutup, kecuali Anda centang "Ingat Saya"). Data script tersimpan privat di akun JSONBin Anda.
