ℹ️ Tentang TamperVault

🛡️ TamperVault v1.0

Penyimpan script Tampermonkey/Greasemonkey berbasis cloud tanpa batas ukuran.

✨ Fitur:

  • ✅ Penyimpanan cloud (JSONBin.io) - bukan Google Sheet, bukan storage PC
  • ✅ Tanpa batas ukuran script
  • ✅ Bisa diakses dari mana saja oleh semua pengguna
  • ✅ Berbagi script dengan pengguna lain via Bin ID
  • ✅ Export/Import JSON
  • ✅ Pencarian & filter script
  • ✅ Dark mode modern

📖 Cara Pakai:

  1. Daftar gratis di jsonbin.io
  2. Ambil Master Key dari dashboard
  3. Klik ⚙️ Setup di aplikasi ini, masukkan Master Key
  4. Mulai simpan script!
⚙️ Setup JSONBin.io
📝 Panduan Setup:
1. Buka jsonbin.io dan daftar gratis
2. Login, lalu copy X-Master-Key dari dashboard
3. Paste di bawah ini. Bin ID akan dibuat otomatis.
4. Untuk mengakses bin orang lain, isi juga Bin ID mereka.
